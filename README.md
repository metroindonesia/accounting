# Accounting System

Accounting system for metroindonesia

Features:
- Chart of Accounts
- Sub Accounts:
	- Partner (Vendor/Customer/Employee)
	- Site
	- Project
	- Department
- Journal Entry
- Journal Voucher
- General Ledger
- Trial Balance
- Profit and Loss
- Balance Sheet
- Cash Flow
- Budgeting
- Fixed Assets
- Inventory
- Sales
- Purchase
- User Management
- Role Management
- Permission Management
